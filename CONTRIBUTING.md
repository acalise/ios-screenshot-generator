# Contributing

Thanks for your interest in contributing! Here's how you can help.

## Reporting Bugs

Open a [GitHub issue](https://github.com/acalise/ios-screenshot-generator/issues) with:

- A clear title and description
- Steps to reproduce the problem
- Expected vs. actual behavior
- Your browser and OS

## Suggesting Features

Open a [GitHub issue](https://github.com/acalise/ios-screenshot-generator/issues) and tag it as a feature request. Describe the use case and why it would be useful.

## Submitting Pull Requests

1. **Fork** the repo and clone your fork
2. **Install dependencies:** `npm install`
3. **Start dev server:** `npm run dev` (runs on localhost:3000)
4. **Create a branch** from `main` with a descriptive name
5. **Make your changes** and test them in the browser
6. **Keep PRs focused** - one feature or fix per PR
7. **Open a Pull Request** against `main` with a clear description of what changed and why

## Code Style

- TypeScript with strict mode
- Follow existing patterns in the codebase
- Use Tailwind CSS utilities for styling
